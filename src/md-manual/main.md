Markdown（MD）是一种轻量级的**标记语言**，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

以下是 Markdown 最常用的核心语法指南，帮助你快速上手：

---

## 1. 标题 (Headings)

在文字前面加上 `#` 号来表示标题，`#` 的数量代表标题的级别（最多支持六级）。**注意：`#` 和文字之间要有一个空格。**

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

```

---

## 2. 字体格式 (Text Styling)

用于强调文本的粗细、倾斜等样式。

```markdown
*这是斜体* 或者 _这是斜体_
**这是粗体** 或者 __这是粗体__
***这是粗斜体*** 或者 ___这是粗斜体___
~~这是删除线~~

```

> **效果：**
> *这是斜体*
> **这是粗体**
> ~~这是删除线~~

---

## 3. 列表 (Lists)

### 无序列表

使用 `*`、`+` 或 `-` 加上空格。

```markdown
- 苹果
- 香蕉
- 橙子

```

### 有序列表

使用数字加上 `.` 再加上空格。

```markdown
1. 第一步
2. 第二步
3. 第三步

```

### 任务列表 (Todo List)

```markdown
- [x] 已完成任务
- [ ] 未完成任务

```

---

## 4. 区块引用 (Blockquotes)

在文本前加入 `>` 号，可以嵌套使用。

```markdown
> 这是一段引用。
> > 或者是嵌套引用。

```

> **效果：**
> 这是一段引用。
> > 或者是嵌套引用。
> 
> 

---

## 5. 代码 (Code)

### 行内代码

使用反引号 ` 包裹代码。

```markdown
使用 `print()` 函数输出内容。

```

### 代码块

使用三个反引号 ``` 包裹，并在第一组反引号后面指定语言（可选，用于代码高亮）。

```markdown

```python
def hello_world():
    print("Hello, Markdown!")
``` \`\`\`


```

---

## 6. 链接与图片 (Links & Images)

这两个语法非常相似，只是图片前面多了一个感叹号 `!`。

### 链接

```markdown
[百度](https://www.baidu.com)

```

> **效果：** [百度](https://www.baidu.com)

### 图片

```markdown
![图片描述/替代文字](https://example.com/image.jpg)

```

---

## 7. 分割线 (Horizontal Rules)

使用三个或以上的 `*`、`-` 或 `_` 来建立一条分割线。**行内不能有其他内容。**

```markdown
---
***
___

```

---

## 8. 表格 (Tables)

使用竖线 `|` 来分隔单元格，使用短横线 `-` 来分隔表头和内容。可以使用冒号 `:` 来控制对齐方式。

```markdown
| 外语缩写 | 对应的中文 | 对齐方式 |
| :--- | :---: | ---: |
| HTML | 超文本标记语言 | 左对齐 (默认) |
| CSS | 层叠样式表 | 居中对齐 |
| JS | 脚本语言 | 右对齐 |

```

> **效果：**

| 外语缩写 | 对应的中文 | 对齐方式 |
| --- | --- | --- |
| HTML | 超文本标记语言 | 左对齐 (默认) |
| CSS | 层叠样式表 | 居中对齐 |
| JS | 脚本语言 | 右对齐 |

---

## 9. 转义字符 (Escaping)

如果你只想打出 Markdown 的符号（比如真的想打出一个 `*` 而不是让文字变斜体），可以在符号前加上反斜杠 `\`。

```markdown
\*这不是斜体\*

```

> **效果：** *这不是斜体*
