# Learning Resource

## 📚 资源列表

### [JavaScript 语法快速入门](javascript-guide.md)

面向有 Python/C 基础的开发者，涵盖：
- 变量声明与数据类型
- 运算符与字符串操作
- 数组与对象操作
- 函数与类（ES6+）
- 异步编程（Promise / async-await）
- 模块化与常用语法糖

### [VSCode 插件推荐](vscode-plugins.md)

精选实用插件，覆盖日常开发全场景：
- 通用必装插件
- AI 效率工具
- 语言专属插件（Python/JS/Go/Rust/Java 等）
- 界面主题与图标
- 开发工具（Docker/Git/测试/远程开发）

### [Git Merge、Rebase 与 PR 合并详解](learn-pr/git-pr-guide.html)

交互式可视化教程，涵盖：
- Git Merge 的工作原理与动画演示
- Git Rebase 的工作原理与逐步演示
- Merge vs Rebase 对比与选择建议
- PR 的三种合并方式：Merge Commit / Squash / Rebase
- 多人协作中的冲突解决策略
- 同步上游变更的最佳实践

## 🎯 使用说明

### 快速开始

1. **JavaScript 学习**：打开 [javascript-guide.md](learn-js/javascript-guide.md)，按章节逐步学习
2. **Git 与 PR 合并**：在浏览器中打开 [git-pr-guide.html](learn-pr/git-pr-guide.html)，使用左右键 ← → 切换步骤学习
3. **配置编辑器**：参考 [vscode-plugins.md](vscode-plugins/vscode-plugins.md) 安装适合你项目的插件

### 本地阅读

```bash
# 克隆仓库
git clone <repository-url>
cd Learning_Resource

# 使用 VSCode 打开
code .
```

## 💡 学习建议

1. **对比学习**：利用已有的 Python/C 基础，对照理解 JavaScript 语法差异
2. **实践为主**：边学边写，在浏览器或 Node.js 环境中验证
3. **工具辅助**：安装推荐的 VSCode 插件提升编码效率
4. **持续更新**：结合 MDN 等官方文档深入学习
